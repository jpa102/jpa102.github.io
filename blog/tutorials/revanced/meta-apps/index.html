Meta apps
revision: 2025年11月11日 12:44:11
IMPORTANT NOTE: Make sure your ReVanced Manager is using the latest patches version before proceeding
Getting the APK File - Facebook & Messenger
  • Go to ReVanced Manager, tap the Patcher tab, then the Select an app, and tap the Suggested: xx.xx.xx box in the app that you want to patch. This should perform a google search for the app name and version, where it should display APKMirror as the 1st search result (The APK version should match with the suggested one that you saw)
  • Once you're in the APKMirror web page, scroll till you find the APK links for Facebook (Don't tap the one that says BUNDLE, it should be nodpi, and it should match your device's architecture)
  • Download the APK file, but DO NOT install it YET. If you do, you will instead install the unpatched Facebook app
  • To prevent installation errors for Facebook & Messenger:
    • Uninstall both the original unpatched Facebook & Messenger
    • Uninstall Meta App Installer, Meta App Manager, & Meta Services (If you can see them in your Phone settings => Apps)
    • Avoid applying Universal patches because they fail: https://github.com/ReVanced/revanced-patches/issues/?q=is%3Aissue%20androlib, this is actually for meta apps as a whole
Getting the APK File - Instagram & Threads
  • Go to ReVanced Manager, tap the Patcher tab, then the Select an app, and tap the Suggested: xx.xx.xx box in the app that you want to patch. This should perform a google search for the app name and version, where it should display APKMirror as the 1st search result (The APK version should match with the suggested one that you saw)
  • Once you're in the APKMirror web page, scroll till you find the links for Instagram. You will have to download the BUNDLE variant this time (doesn't apply for older instagram releases)
  • Using Antisplit-M, provide the apkm file (rename the apkm file name though, there are reports that having apkmirror.com.apkm makes it fail to be selectable)
  • You should have a patchable APK file after that process
Now to the patching process
  • Go to ReVanced Manager, tap the Patcher tab, then the Select an app, and tap the "Storage" button. Now find the APK that you just downloaded / converted using Antisplit-M
  • If you want, you can review the patches by tapping the Selected patches section. Changing the patches selection requires enabling Allow changing patch selection at Settings
  • Tap the "Patch" button and wait / watch the patching process
  • If the patching process is successful, you can now install the patched app
Questions and Answers
> I can't download anything from APKMirror
  • Find a reputable source for the base APK file, APKPure for example
> Why I can't patch a meta app with any Universal patches?
  • It's because of the androlib issue that's present: https://github.com/ReVanced/revanced-patches/issues/?q=is%3Aissue%20androlib
> It says "Installation conflict"
  • Did you just recently uninstalled and reinstalled ReVanced Manager? If so, then the keystore being used is now different from the last time the app was patched & installed (the root problem: signature mismatch)
> It says "Can't downgrade"
  • You have patched an older (but compatible) version of the app and it's trying to install it on top of the greater one (Example: 490.0.0.63.82 to 538.0.0.53.70)
> I was told to repatch after posting a question to my problem but I still experience it, what can I do?
  • You can open a bug report at https://github.com/ReVanced/revanced-patches/issues
  • Or open an issue / discussion at my repository so we can have a nice little chat: https://github.com/jpa102/revanced-discussions (i may not be able to see your discussion immediately though as I tend to be offline at some times)
> I can't successfully patch using ReVanced Manager on my device, what can I do?
  • If you have access to a PC, you can still patch using revanced-cli [ sample command: java -jar revanced-cli-5.0.1-all.jar patch facebook-490.0.0.63.82-apkmirror.apk -p patches-5.46.0.rvp ]. You will need to set up the Java environment first though in order to use it (follow the pc guide in here: /blog/tutorials/revanced/general-install/)
  • Or if you have a secondary device or someone else's device that you can borrow, you can try patching from there and export the patched apk to your device afterwards
